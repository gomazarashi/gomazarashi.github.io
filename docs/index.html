about

このサイトについて

技術記事、翻訳、作成した資料、個人開発の内容をまとめるための個人サイトです。

自サイト内の記事に加えて、外部媒体の執筆物や関連資料への導線もここに集約します。

構築環境

  • コンテンツ・マークアップ: Typst
  • 静的サイト生成: Tola
  • 開発環境: Nix
  • ホスティング・デプロイ: GitHub Pages

Repository: github.com/gomazarashi/gomazarashi.github.io
